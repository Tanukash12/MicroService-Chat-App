# MicroService Chat App

A scalable real-time chat application built using a **microservices architecture**.  
This project demonstrates how modern backend systems can be designed using **independent services and modular architecture** using **Node.js and TypeScript**.

---

### 🚀 Features

- Microservices-based backend architecture
- Modular and scalable system design
- Email service integration
- TypeScript for type safety
- Independent services for better maintainability

---

### 🏗️ Architecture

This project follows a **microservices architecture**, where each service handles a specific functionality such as messaging, authentication, or notifications.

Example structure:
```
backend/
├── auth-service
├── chat-service
├── mail-service
└── gateway
```


Each service can run independently and communicate via APIs.
